# Сборник рецептов «Здоровая еда» — Андрей Мокич (2025)

По рекомендациям нутрициолога Марины Елисеевой (@tri.daily, Gina Nutrition).

Полный сборник: 82 страницы, 120 000+ символов текста — импортирован из PDF.
Содержит меню на 2 недели с разбивкой по приёмам пищи.

## Структура сборника

### Раздел: Завтраки
Примеры блюд:
- Киноа с зелёной спаржей, яйцом пашот и салатом из авокадо, рукколы и шпината
- Гречка с печёным лососем, салатом и соусом из авокадо и песто
- Овсяная каша с фруктами и семечками
- Крученый омлет с салатом из булгура и лаймом
- Кускус с цитрусами, бананом и мятой
- Каша из киноа с апельсиновым соком и ягодами

### Раздел: Обеды
(в сборнике — соответствующие рецепты с балансом БЖУ)

### Раздел: Полдники
(в сборнике — лёгкие приёмы пищи с белком и овощами)

### Раздел: Ужины
(в сборнике — белковые ужины)

## Принципы:
- Сбалансированное здоровое питание для всей семьи
- Не для снижения веса (но совместимо с программой похудения при соблюдении порций)
- Без учёта индивидуальных диет/заболеваний — при наличии рекомендаций врача консультироваться отдельно
